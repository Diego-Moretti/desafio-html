Tags "h1 até a h6" <h1>

As tags <h1></h1> até <h6></h6> são usadas para definir diferentes níveis de títulos em uma página HTML. Elas representam cabeçalhos ou títulos, e a principal diferença entre elas está no tamanho e importância. Essas tags são importantes porque não apenas dão uma estrutura clara ao seu conteúdo, mas também ajudam os motores de busca e leitores de tela a entenderem a hierarquia e a importância do texto na sua página. Em resumo, <h1> é o título mais importante, <h2> é menos importante que <h1>, e assim por diante. É como organizar um livro com capítulos e subcapítulos.

Tag <p>

A tag <p></p> é usada para definir parágrafos em HTML. Ela é fundamental para estruturar o conteúdo textual da sua página. Cada parágrafo começa com <p> e termina com </p>. Todo o texto entre essas duas tags é considerado como parte do mesmo parágrafo. Portanto a tag <p> é uma ferramenta simples, mas crucial para organizar e estruturar o texto em uma página HTML, tornando-a mais compreensível para os visitantes e facilitando a formatação e estilização do conteúdo.

Tag <mark>

A tag <mark></mark> é usada para destacar ou marcar parte do texto em HTML. Ela é frequentemente usada para chamar a atenção do leitor para informações específicas.

Tag <small>

A tag <small></small> é usada para diminuir o tamanho do texto em HTML. Ela é frequentemente utilizada para informações secundárias, notas de rodapé ou qualquer texto que seja menos importante em comparação com o conteúdo principal.

Tag <i>

A tag <i></i> é usada para colocar texto em itálico em HTML. Ela é frequentemente usada para destacar palavras ou frases de uma maneira que indica um estilo diferente, sem enfatizar a importância.

Tag <u>

A tag <u></u> é usada para aplicar sublinhado a um texto em HTML. Ela é utilizada para destacar ou indicar que o texto é importante, embora o uso de sublinhado possa variar dependendo do contexto e das práticas de design modernas.

Tag <strong>

A tag <strong></strong> é usada para enfatizar um texto tornando-o mais forte ou importante, geralmente sendo renderizado como texto em negrito.

Tag <ol>

A tag Tag <ol></ol> é usada para criar listas ordenadas em HTML. Listas ordenadas são aquelas em que os itens são numerados ou marcados com algum tipo de ordem. Essa tag é usada para criar listas ordenadas, onde a ordem dos itens é importante. Por padrão, os itens são numerados, mas isso pode ser personalizado com CSS.

Tag <ul>

A tag <ul></ul> é usada para criar listas não ordenadas em HTML. Listas não ordenadas são aquelas em que os itens não têm uma ordem específica e geralmente são marcados com pontos, traços ou outros tipos de marcadores. é usada para criar listas não ordenadas, onde a ordem dos itens não é importante. Os itens geralmente são marcados com pontos, traços ou outros símbolos.

Tag <li>

A tag <li></li> é usada para definir cada item de uma lista, seja ela ordenada <ol> ou não ordenada <ul>. Cada <li></li> envolve o conteúdo específico do item.

Tag <a>

A tag Tag <a><a> é usada para criar hiperlinks em HTML, permitindo que os usuários cliquem em um texto ou imagem para navegar para outra página ou recurso na web. Além do href, você pode usar outros atributos, como target para especificar se o link será aberto em uma nova janela ou na mesma janela.

Tag <hr>

A tag <hr> é usada para criar uma linha horizontal em HTML, geralmente usada para separar conteúdos ou criar uma divisão visual na página.

Tag <sub>

A tag <sub></sub> é usada para criar texto sobrescrito em HTML, o que é útil para representar números ou caracteres que aparecem abaixo da linha de texto normal.

Tag <sup>

A tag <sup></sup> é usada para criar texto sobrescrito em HTML, sendo útil para representar números ou caracteres que aparecem acima da linha de texto normal.

Tag <blockquote>

A tag <blockquote></blockquote> é usada para criar uma seção de citação em bloco em HTML, indicando que o texto dentro dela é uma citação retirada de outra fonte.

Tag <del>

A tag <del></del> é usada para indicar texto que foi removido ou deletado. Ela é geralmente utilizada em contextos onde se deseja mostrar que uma parte do texto foi removida ou é obsoleta. Exemplo texto desatualizado.

Tag <font>

A tag <font> era usada anteriormente em HTML para definir o estilo de fonte de um trecho de texto. No entanto, ela foi declarada obsoleta em HTML5, e é recomendado o uso de estilos CSS para a formatação de fontes em vez da tag <font> .
